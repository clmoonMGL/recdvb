# recdvb

Based on recdvb published on http://cgi1.plala.or.jp/~sat/


## how to install

```bash
 $ ./autogen.sh
 $ ./configure [--enable-b25]
 $ make
 $ sudo make install
```


## tested environment

- Device: PT3(with kernel 4.9.6 earth_pt3 driver)
- Software: Mirakurun
